# 知识分享

## 修改流程

推荐使用 `pnpm` 包管理工具
1. 安装[node](https://nodejs.cn/download/)>=16.0.0
2. 使用 `npm install -g pnpm` 安装
3. 克隆本仓库代码 `https://github.com/hust-404/hust-404.github.io.git`
4. 运行`pnpm install` 初始化项目
5. 本地修改完成之后运行 `pnpm docs:dev`，确保没有问题再提交
